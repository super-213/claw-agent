# Skill: calculator

## 功能
执行数学表达式计算。

## 输入格式
用户输入数学表达式，如 "2+3*4"、"sqrt(16)"、"pow(2,10)"。

## 输出要求
- 先显示原表达式
- 再显示计算结果
- 保留 2 位小数（如适用）
- 格式：`[计算] 表达式 = 结果`

## 输出格式
使用 [计算] 标记开头，如：
[计算] 3-8*7 = -53

完成后无需额外标记，系统会自动识别。

## 支持运算
+ - * / // % ** () 
函数：abs, round, pow, min, max

## 示例
输入: 2+3*4
输出: [计算] 2+3*4 = 14

输入: pow(2,10)
输出: [计算] pow(2,10) = 1024
